Featured Publications

1. Feature Evaluating Learning

IEEE transactions on pattern analysis and machine intelligence 2017
One-Pass Learning with Incremental and Decremental Features

C Hou, Z Zhou.

  • This work explores a dynamic learning problem that the feature space evolves over time, with some features being vanished and some other features being augmented. We compress important information of vanished features into functions of survived features, and then expand to include the augmented features to tackle this problem.
IEEE transactions on pattern analysis and machine intelligence 2018
Safe Classification with Augmented Features

C Hou, L Zeng, D Hu.

  • This work proposes a secure method to handle the challenge that augmented features may not bring positive effect for Multi-view learning tasks. We learn various classifiers by employing a specially designed robust loss and integrate all classifiers by approximately maximizing the performance improvement.
IEEE transactions on pattern analysis and machine intelligence 2023
Incremental Learning for Simultaneous Augmentation of Feature and Class

C Hou, S Gu, C Xu, Y Qian.

  • This work explores a dynamic learning problem that both feature and label space appear a simultaneous increase over time. We add a regularizer to guarantee the reusability of the model trained on previous data. Then we extend our method to multi-shot tasks.

2. Streaming Learning

IEEE transactions on pattern analysis and machine intelligence 2024
Adaptive Learning for Dynamic Features and Noisy Labels

S Gu, C Xu, D Hu, C Hou.

  • This work proposed an adaptive learning method capable of handling dynamic feature addition and reduction with noisy labels. By leveraging optimal transport and incorporating consistency constraints, heterogeneous models facilitate the estimation of the noise transition matrix and the training of classification models, with theoretical guarantees provided.
IEEE Transactions on Knowledge and Data Engineering 2023
Online Learning With Incremental Feature Space and Bandit Feedback

S Gu, T Luo, M He, C Hou.

  • This work introduces a set of algorithms for learning from data streams with incremental feature spaces and bandit feedback. The algorithms theoretically analyze and provide bounds on the cumulative loss.
IEEE Transactions on Knowledge and Data Engineering 2025
Adaptive Learning in Imbalanced Data Streams With Unpredictable Feature Evolution

J Tu, X Tang, S Gu, Y Dai, R Fan, C Hou.

  • This work proposes the ALIU algorithm to handle imbalanced data streams with unpredictable feature evolution via gradient reweighting, robust loss design, model reuse, and ensemble methods, supported by theoretical regret analysis and experimental validation

3. Weakly-supervised Learning

KDD '24: Proceedings of the 30th ACM SIGKDD Conference on Knowledge Discovery and Data Mining 2024
Label Shift Correction via Bidirectional Marginal Distribution Matching

R Fan, X Ouyang, H Tao, C Hou.

  • This paper proposes a method called BMDM that corrects label shift by bidirectionally matching the marginal distributions of labels and features, ensuring stable estimation of target label distributions and improving model generalization, with theoretical proof and extensions to continuous label shift scenarios.
IEEE Conference on Computer Vision and Pattern Recognition (cvpr) 2025
Label Shift Correction via Bidirectional Marginal Distribution Matching

Y Dai, S Gu, R Fan, C Xu, C Hou.

  • This work proposed a method to ensure consistent adaptation to online label shift, with the regret bound provided.
Journal of Machine Learning Research 2023
Label distribution changing learning with sample space expanding

C Xu, H Tao, J Zhang, D Hu, C Hou.

  • This work formulates a novel LDCL framework to solve the problem of Label Distribution Changing Learning with Sample Space Expanding, together with its theoretical guarantee with generalization error bound.
IEEE Transactions on Neural Networks and Learning Systems 2025
Confidence-Based PU Learning With Instance-Dependent Label Noise

X Tang, C Xu, H Tao, X Ma, C Hou.

  • This paper addresses the critical challenge of instance-dependent label noise in positive-unlabeled learning. A robust framework is proposed to quantify uncertain noise through confidence measures while establishing instance-dependent noise modeling.

4. Learning Theory & AI for Science

Applied Mathematics and Computation 2024
An adaptive energy-based sequential method for training PINNs to solve gradient flow equations

J Guo, H Wang, C Hou.

  • This work utilizes one of the gradient flow equations’ significant physical properties, the energy dissipation law, to propose an adaptive energy-based sequential training method for PINNs.
KDD '25: Proceedings of the 31st ACM SIGKDD Conference on Knowledge Discovery and Data Mining 2025

C Xu, X Tang. H Tao, C Hou.

  • This work focuses on the problem of feature-incremental learning, we distill four fundamental strategies—feature tailoring, data adaptation, model reuse, and data reconstruction. Based on these, we provide a theoretical analysis of their generalization ability and conduct a systematic comparative study to derive rigorous, quantitative insights.
IEEE Transactions on Knowledge and Data Engineering 2023
Imbalanced Clustering With Theoretical Learning Bounds

J Zhang, H Tao, C Hou.

  • This work is motivated by the framework of excess clustering risk, we introduce an imbalanced clustering algorithm and establish a corresponding learning-theoretic bound.

Joint embedding learning and sparse regression: A framework for unsupervised feature selection

C Hou, F Nie, X Li, D Yi, Y Wu.

Multiple view semi-supervised dimensionality reduction

C Hou, C Zhang, Y Wu, F Nie.
